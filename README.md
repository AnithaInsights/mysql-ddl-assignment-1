# MySQL Assignment 1 – DDL Commands & Constraints

## Description
This project contains SQL scripts demonstrating MySQL DDL (Data Definition Language) commands and constraints.

## Topics Covered
- CREATE DATABASE
- CREATE TABLE
- ALTER TABLE
- RENAME TABLE
- TRUNCATE TABLE
- DROP TABLE
- DROP DATABASE

## Constraints Used
- PRIMARY KEY
- FOREIGN KEY
- NOT NULL
- UNIQUE
- CHECK
- AUTO_INCREMENT
- DEFAULT
- ENUM

## Tools Used
- MySQL Workbench
- MySQL 8.x
